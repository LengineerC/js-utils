# format

## 函数 - format

使用占位符格式化字符串。 支持三种占位符形式： 
1. `{}`      : 自动按数组顺序取值 
2. `{0}`, `{1}` : 数字索引 (数组) 
3. `{name}` : 对象属性

### 参数

- `str`: `string` - 要格式化的字符串

- `args`: `Record<string, any> | any[]` - 替换占位符的值，数组或对象

### 返回值

`string` - 格式化后的字符串

### 示例

```typescript
format("Hello {}, {1}, {name}", [10, 20]); // 输出: "Hello 10, 20, {name}" （对象占位符不会生效，因为传入的是数组）
```
