# range

## 函数 - range

生成一个包含指定范围数字的数组，类似Python。

### 参数

- `start`: `number` - - 起始值（包含）。如果只提供一个参数，则视为 end，start 默认为 0。

- `end`: `number` - - 结束值（不包含）。

- `step`: `number` - - 步长，默认为 1。可以为负数。

### 返回值

`number[]` - 包含数字序列的数组。

### 示例
```ts
range(5)  // [0, 1, 2, 3, 4]
range(2, 10, 2)  // [2, 4, 6, 8]
range(5, 0, -1)  // [5, 4, 3, 2, 1]
```

### ⚠️ 警告
- 使用浮点数步长会有精度问题，最好只使用整数
